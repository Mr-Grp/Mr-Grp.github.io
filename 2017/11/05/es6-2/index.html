1.1 es5&es6 比较

1.es5超过两个字节不识别

console.log('s',\u20BB7);
s ₻7

会把超过两字节的字符作为两个字符处理。

2.ES6识别大于两字节的字符

console.log(‘s’,\u{20BB7});
𠮷

第一个为4字节字符，但第二个仍为两个字节
