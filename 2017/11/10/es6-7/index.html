1. Promise

异步编程

调用程序时，返回一个 Promise 实例 retrun new Promise(function(resolve,reject){})

当调用这个实例的 then 方法，传递两个回掉后，执行实例内容。

多个连写

1.1 异常捕获

调用then方法后可以使用 .catch(回掉函数) 方法进行异常捕获

1.2 Promise.all

Promise.all([]).then()
将多个实例返回的 Promise 当作一个处理，所有都加载完才会执行。

1.3 Promise.race

Promise.race([]).then()
将多个实例返回的 Promise 当作一个处理，有一个加载完就会执行。

2. Iterator

迭代器
每当调用 for of 的时候会调用 Iterator 进行遍历

数组中手动调用 Iterator 的方法

arr[Symbol.iterator]() 调用迭代器
next() 方法返回map，包括 value 和 done（表示是否结束）

2.1 对象自定义 Iterator

在对象中定义[Symbol.iterator](){}方法
返回值为 next() 方法
next 方法返回一个 包括 value 和 done 的对象，并让索引后移一位，如果已经结束，done 为 true

3. Generator

function 后面加 *
函数里面使用 yield 声明

实例后停留在第一个 yield 前
每次 next 执行下一个 yield 返回一个 包括 value 和 done 的对象，执行到最后一个时，done变为true ，和Iterator 不同的是， Iterator到最后一个的下一个才会变为true
