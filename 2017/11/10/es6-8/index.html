1. Decorator

修饰器,是一个函数，用来修改类的行为

需要安装一个插件
并在.babelrc中添加
"plugins":["transform-decorators-legacy"]

1.1 修饰方法

定义 function(target,name,descriptor)
target:修饰的对象
name:修饰方法名
descriptor：控制行为，返回

方法上 @修饰器函数名

1.2 修饰类

target 类本身

1.3 第三方库修饰器的js库

core-decorators;
npm install core-decorators

2. 模块化

2.1 引入

import {a,b,c} from ...
可选择导入项

如果需要全部导入
import * as lesson from ...
全部导入在lesson对象中

如果导出未命名，可以自己随意取名
import xx from xxx

如果只加载模块
import xxx

2.2 导出

不命名，由引入方去命名。
